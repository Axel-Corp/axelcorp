Logiciel SafePass

SafePass est un gestionnaire de mots de passe sécurisé permettant de stocker et gérer vos identifiants en toute simplicité.

Fonctionnalités Principales :

  • Stockage sécurisé : Cryptage des mots de passe avec un algorithme robuste.
  • Génération automatique : Création de mots de passe forts et uniques.
  • Autocomplétion : Remplissage automatique des identifiants sur les sites web.
  • Authentification multi-facteurs : Sécurisation supplémentaire via un second facteur.
  • Accès multi-appareils : Synchronisation sécurisée sur plusieurs dispositifs.

Technologies Utilisées :

  • Python : Langage de programmation principal.
  • Flask : Framework web pour l'interface utilisateur.
  • SQLite : Base de données pour le stockage des identifiants.
  • AES-256 : Algorithme de cryptage pour protéger les données.

Avantages :

  • Protection optimale des données sensibles.
  • Facilité d'utilisation avec une interface intuitive.
  • Accès rapide et sécurisé aux mots de passe enregistrés.
